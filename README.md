# 简介

vue3-element-plus-admin 是一个基于 Vue3、Vite、ElementPlus、Pinia、VueUse 等的免费中后台模版，它使用了最新的前端技术栈，开箱即用的模板，也可用于学习参考。

# 预览

https://monsterxwx.github.io/vue3-vite-admin

# 技术栈
- vue3
- vite
- element-plus
- vue-use
- unocss
- vue-router
- scss
- pinia
- eslint
- stylelint

# 使用

```bash
pnpm i
```

# 启动

```bash
pnpm dev
```
